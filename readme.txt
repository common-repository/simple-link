== Description ==

This is a simple plugin that enables you to add hyperlinks to a post on selected text with just a click! 
Also you can easily find certain post that you want to add hyperlink to using the searching feature supplied by the plugin.
It works well with the default editor of WordPress as well as FCKeditor.

The next release will contain an option page. You can configurate basic settings such as how many post buttons to display, etc.

== Installation ==
= Method 1 =
1. Go to Plugins of your WordPress admin site.
2. Click `Add New`.
3. Type in `Simple Link` and click `Search Plugins`.
4. Find Simple Link and click `Install Now`.
5. Activate Simple Link after installation finishes.
6. Come to write a new post and enjoy the convenience of `Simple Link`.

= Method 2 =
1. Upload the whole directory `simple-link` to the `/wp-content/plugins/` directory.
2. Follow step 5 and 6 in Method 1.

== Frequently Asked Questions ==

= 1.Why the plugin doesn't work? =

Are you using FCKeditor? If you are, please update your FCKeditor to a version greater than 2.5.0.
Then Simple Link will work great!
If it doesn't help, contact me via [email](mailto:i@renzuocheng.com).

= 2.Will hyperlinks created by Simple Link be removed if I delete the plugin? =

No. You can feel free to remove Simple Link if you feel you don't like it. But I believe you will get addicted to it.
